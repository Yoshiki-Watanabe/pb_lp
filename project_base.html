Project BASEとは、一つの会社に依存せず、プロジェクト単位でジョブホッピングする新しい働き方を模索するプロフェッショナル人材と、プロジェクトに必要な人材を探している企業様をつなげるプラットフォームサービスです。

2008年の創業から現在まで、数百件に及ぶマッチングを成立させてきました。

その経験から、個々のキャリアビジョンに合った、キャリアアップにつながるプロジェクトをご紹介することが可能です。

Project BASEの特徴

キャリアアップに繋がる案件

キャリアビジョンに基づいて、あなたのキャリアアップに繋がる案件が見つかります。

契約が途切れにくい

案件従事中に営業を依頼することで、契約が途切れることなく継続した稼働ができます。

高額案件が見つかる

今年で9周年を迎える実績を持つProject BASEならではの、高額案件を受注できます。

20代 男性/フリーコンサルタント

夢だったフリーコンサルタントの道へ

コンサル案件を紹介してくれるサービスを探していた時に偶然見つけました。実際に登録してみようと思ったのはProject BASEなら私のキャリアを長期的な目線から支援してくれるのではないかと思ったからです。

実際にProject BASEを利用してみて、事前のヒヤリングから企業との条件交渉、さらには業務期間中のサポートまで一貫して行ってくれたのは非常に助かりました。

具体的な仕事内容としては大規模なシステム改修案件のプロジェクトリーダーとして主に設計部分を担当しました。これまでに経験したことないくらい大きなシステムの設計だったため、今後のキャリアアップとして最適な案件を紹介していただけて感謝しています。

子育てしながらでもキャリアアップ

ちょうど1年前に子供が生まれたのをきっかけに前職の外資系コンサルティング会社を退職しました。復職も考えましたが、もっと自由に働きたいと思っていたためフリーのコンサルタントとしての仕事を探していました。

Project BASEの担当者の方には、子育てと仕事を両立したい私のために時間的制約の少ない案件を探していただきました。企業とコンサルタント側の要望をしっかりと聞いたうえでマッチングしてくれるのが良かったです。

前職の経験を活かして、中小企業の戦略策定の支援をしています。今後もProject BASEを活用させていただき、子育てをしつつ、キャリアも磨いていけたらと思っています。
